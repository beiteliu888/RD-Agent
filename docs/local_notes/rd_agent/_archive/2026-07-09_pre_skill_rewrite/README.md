## 写作规则

- 中文为主；命令、路径、配置项、技术名词保留英文。
- 使用 `【事实】`、`【推断】`、`【建议】`、`【待验证假设】` 标注判断强度。
- 不把 README、论文或 demo claim 直接当成已验证事实。
- 不配置真实 API key、账号、cookie、token 或云服务。
- 不运行涉及 LLM API、Docker 拉镜像、Qlib 数据下载、Kaggle、Hugging Face、训练或上传的流程，除非用户明确批准。
